
- 신규: RN번호가 포함된 신규 메일

- 처리중: 작업자가 할당된 상태

- 서류미비 요청: EV 보완이 오기 전 작업자가 서류미비에 대해 요청 상태

- 서류미비 도착: 요청한 서류미비가 도착한 상태

- 추후 신청: 당일 신청이 아닌 '정해진 날짜'에 넣는 건

- pdf 전처리: 작업자가 pdf 작업을 완료해 저장한 상태

- 첨부파일 누락: 신규 메일인데 첨부파일이 없는 상태 -> 자동 메일 회신

- EV보완 요청: EV에서 보완요청이 온 상태

- 서류미비 요청_EV: EV 보완 사항에 대해 DA에게 서류미비를 요청한 상태

- 서류미비 도착_EV: EV 보완 사항에 대해 요청한 서류미비 메일에 대해 서류가 도착한 상태

- EV보완 완료: 작업자가 EV 보완 완료를 처리한 상태 -> 자동으로 확인 후 '처리완료'로 넘어감

- 확인필요: 팀장 or 매니저가 확인이 필요한 상태

- 처리불가: 신청이 들어가면 안되는 건 -> 이 상태에서는 신청 X

- 취소: 현재 취소가 된 상태(재신청은 포함 안됨)

- 중복메일: 처리한 RN번호와 동일한 메일이 추가로 온 상태

- 처리완료: EV 신청완료 상태이면서 (DA/DOA/EV지자체로부터) 추가 요청이 없어 작업할 필요가 없는 상태

* 빨간 행: '긴급' 건으로 다른 건보다 우선으로 신청을 넣어야 하는 상태
